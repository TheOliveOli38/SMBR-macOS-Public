# Super Mario Bros Remastered
A Remake / Celebration of the original 'Super Mario Bros.' games. Features new levels, custom modes, new characters, alongside a full level editor / custom level system!

## Unofficial macOS Port
Changes made in this repo have been integrated into yuriko-shimizu's [Super Mario Bros. Remastered Public (Mac)](https://github.com/yuriko-shimizu/Super-Mario-Bros.-Remastered-Public-Mac) repository. To download the latest release of SMBR for Mac, please refer to that repo instead.

## German localization
Work on SMBR's German localization can be found in the [de-loc](https://github.com/TheOliveOli38/SMBR-macOS-Public/tree/de-loc) branch of this repository.
